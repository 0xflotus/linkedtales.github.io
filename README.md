# ![LinkedTales](https://raw.githubusercontent.com/linkedtales/linkedtales.github.io/master/logo.png) [![Gitter](https://badges.gitter.im/linkedtales/community.svg)](https://gitter.im/linkedtales/community?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge)

LinkedTales is a non-profitable project which uses data from the best professionals worldwide for educational purposes only to extract helpful insights for our career development!

## Heeelp!
Ideas, suggestions, and design work are appreciated, [please open an issue](https://github.com/linkedtales/linkedtales.github.io/issues/new) or [meet us at gitter](https://gitter.im/linkedtales/community) to get in touch!

## We care about privacy and third-party terms of use.
Personal information isn't stored anywhere to guarantee anonymity, privacy and for not violating any third-party terms of use, we only use the statistical algorithms on demand from data which is publicly available.
